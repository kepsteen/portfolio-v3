## Overview

This site is the third iteration of my portfolio. It leans into a terminal-inspired layout, monospace typography, and a theme switcher so visitors can try different DaisyUI palettes.

## Highlights

- App Router with React Server Components where it keeps payloads small.
- MDX-backed content for experience and projects so copy and structure stay in one place.
- Responsive navigation with a drawer for secondary links and theme controls.
